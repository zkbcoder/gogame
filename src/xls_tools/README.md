# EXCEL导出json文件

| id   | id             | type | num  |
| ---- | -------------- | ---- | ---- |
| 备注:  | 消耗id           | 消耗类型 | 消耗数量 |
| int  | int            | int  | int  |
|      | "cost_thing":{ |      | }    |

* 第一行为字段名
* 第二行为备注，程序不读
* 第三行为类型[int, string目前只支持这两种]
* 第四行用来做内嵌子结构用，如果有则需要{}成对出现，可以多个嵌套
